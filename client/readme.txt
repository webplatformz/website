This is the client side of the website.

Installation
------------

Install the build tools (once):
npm install -g bower gulp

Install the dev dependencies required for the build chain (once):
npm install

Install the frontend app dependencies (once):
bower install


Important files
---------------
package.json - contains the dev dependencies for the build chain
bower.json - contains the frontend app dependencies, e.g. angular, bootstrap, etc.
.bowerrc - defines the target directory for the frontend app dependencies
gulpfile.js - defines the build chain
